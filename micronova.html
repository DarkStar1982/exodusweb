Cubesat Simulator

miroNOVA

microNOVA is a cubesat development kit that contains all necessary hardware and software for early prototyping, ground testing and validation activities for nanosatellite projects, packed in a compact 1U (10x10x10 cm) chassis. Write, test and develop code that is ready for the real space missions!

Software

  • Linux Operating System
  • ESA Nanosatellite MO Framework
  • NOVA Mission Software Development Kit






Hardware

  • Primary Onboard Computer (ARM)
  • 9-axis Inertial Measurement Unit
  • GPS receiver
  • Software Defined Radio receiver
  • Imaging Camera
  • Battery pack for autonomous operations

HARDWARE FEATURES

PRIMARY ONBOARD COMPUTER

microNOVA primary onboard computer is a Raspberry Pi, allowing you to use already familiar development tools to build space applications on day one.

SOFTWARE DEFINED RADIO

We included SDR hardware in the box, so you can practice working with signal processing applications, from space and ground sources.

VISIBLE LIGHT CAMERA

Imaging camera is one of the most common cubesat instruments, so we included that one as well. Try taking photos on the ground first - so you will be ready to make them in space!

INERTIAL MEASUREMENT UNIT

Inertial Measurement Unit is the key part of any satellite attitude control subsystem. Learn the basics of satellite flight dynamics. Develop and test your ADCS algorithms on the ground. Includes 3-axis accelerometer, magnetometer and gyroscope.

GPS RECEIVER

Did you know that GPS navigation works in space as well? Many cubesat missions used off-the-shelf GPS modules to help them navigate in orbit. So we included this hardware part to make our hardware platform complete.

FULL EXPANDABILITY

Our system is using on well known, easy to use hardware parts, offering I2C, UART and SPI communications buses, which means adding more hardware to microNOVA is simple and fun.

SOFTWARE FEATURES

"NOVA" SOFTWARE DEVELOPMENT KIT

Any hardware is useless without proper software that can support all its features. That's why the primary value of microNOVA lies in its SDK, which will allow you to seemlessly access all the functionality. And there is more - we aim to launch our own CubeSat mission and will give you the chance to run your software on our satellite. Read more about this mission: NOVA mission.

...AND MORE SOFTWARE TOOLS

You will not be limited by software we provide. We will include additional software packages that will allow you to create applications in many other areas. Prototype IoT products? Create a satellite ground station? Mount microNOVA on a drone? Your imagination is the only limit!
